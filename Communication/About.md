## 🚀 About

This folder contains **basic communication system simulations implemented using GNU Octave / MATLAB**.

The scripts are designed to help students understand **core concepts of analog and digital communication** through **simple, well-documented simulations**.

These simulations help learners move from:

**Theory → Visualization → Intuition**

They are especially useful for **ECE students studying Communication Systems**.

---

## 🧠 Concepts Covered

The simulations in this folder demonstrate the following key topics:

* 📶 **Amplitude Modulation (AM)**
* 📡 **Frequency Modulation (FM)**
* 🔄 **Phase Modulation (PM)**
* ⏱️ **Sampling Theorem**
* 🌊 **Signal Reconstruction**
* 🎚️ **Filters**

  * Low Pass Filter (LPF)
  * High Pass Filter (HPF)
* 🔊 **Additive White Gaussian Noise (AWGN)**
* 📉 **Envelope Detection**

---

## 🎯 Learning Goals

These simulations will help you:

* Visualize modulation techniques
* Understand sampling and reconstruction
* Observe noise effects on signals
* Learn basic filtering techniques
* Build intuition for real communication systems
